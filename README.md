# Single-Page CV

A simple one-page CV built with semantic HTML.  
Includes SEO tags, Open Graph tags, and a favicon.

## Live Project URL
http://127.0.0.1:5500/index.html

## Features
- Semantic HTML structure
- Sections for skills, education, and experience
- SEO meta tags for better search visibility
- Open Graph tags for social sharing
- Favicon included

## How to Use
Open `index.html` in any browser to view the CV.

## Project Structure
index.html  
favicon.png (or your favicon file)

## Purpose
Practice building clean, accessible, well-structured HTML for a personal CV page.
